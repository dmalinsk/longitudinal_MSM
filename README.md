# longitudinal_MSM
Code for the estimation of a longitudinal marginal structural model (MSM) with IPW

This is code related to the following paper:

"Marginal Structural Models for Quantifying the Causal Effects of Exposure to Ambient Air Pollution on Progression of CT Emphysema in the MESA Lung and MESA Air Studies" by D. Malinsky, M. Wang, R. Heise, C.L. Pistenmaa, E.A. Hoffman, L. Shepphard, A.A. Szpiro, A. Laine, E. Angelini, B. Smith, J.D. Kaufman, and R.G. Barr, published in the American Journal of Epidemiology (2025).

See the manuscript (and supplementary materials) for important details. Access to the data in the paper requires request to the MESA collaboration: https://mesa-nhlbi.org/

Contact: d.malinsky@columbia.edu
